문제 2 )
	외부 요인에 영향받지 않는 제주 관광 산업을 만들어 볼까!!!

- 사드영향 등 외부요인 고려 x
- 

- 
제주 주요 3대 산업 현황 비교 -> 지능형 관광콘텐츠 
-> 방문객 조사(내국인 / 연령층 / 성별 / 목적 ) 
-> 최근 미디어 동향 -> 설문조사( 불만족 이유 )
-> 
 

제주 현황 :
- 관광업체 등록 회사 목록 ( 분류 )
- 제주도 관광 컨텐츠 ( 음식, 자연, 액티비티 ) - 방문 이유 data
- 방문객 조사( 내국인 / 연령층 / 성별  ) data
- 카드 매출 조사 
- 자동차 현황( 렌트카 ) 
- 게스트 하우스 
- 클린 하우스
- 시티 투어 버스 
- 관광(여행) 루트
- 제주 지역총생산 data 
- 불친절한 서비스(설문조사) data
- 유튜브 영향력 data 
- 제주 지역 주력산업 현황( 지능형 관광 콘텐츠 ) data
- 

조사 방향 :
- 중국 이외의 외국인 도입 유치 필요
- 홍보물품 비치
- 제주의 특색 유지 방향
- 장기간(1~2개월) 여행 및 정착 지원   ㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇ
- 인플루엔서와 함께 여행
- 유튜브 개인 컨텐츠 제작 활동( 소재가 제주도 )  ㅇㅇㅇㅇㅇㅇㅇ
- 유튜버 + 장기간 여행 및 정착 지원 ==> 홍보  ㅇㅇㅇㅇㅇㅇㅇㅇ
- 스마트 관광(AR) 
- 제주 특색을 가진 생필품
-
